Display

O dispaly serve para dizer como e se o elemento vai aparecer. Todos os elemnetos já vem com display por padrao.

  • block: display que ocupa toda a linha da página. Algumas tags vem com esse estilo por padrão como div, h1-h6, p, forms, reader, footer, section.


  • inline: o elemento ocupa a largura nescessária. Tags a, span e img.


  • none: os elementos não aparecem, mas continuam ativos. Ex:

    Clique aqui.



  • inline-block: diferentes conteúdos podem ficar na mesma linha ocupando o espaço dos seus elementos. Nesse display, differente do inline as propriedades width e height funcionam.
