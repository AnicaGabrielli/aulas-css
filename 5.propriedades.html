Propriedades html

  • Color: cor do texto; hex, rgb, nome

  • background: fundo, imagem...EX:

    • Div com imagem de fundo.

  • border: cor, textura e grossura de bordas; Existem vários estilos de borda: solid, dotted(pontinhos), deshed(tracinhos), double(borda dupla), groove(3d), ridge, inset(por dentro), outset(por fora), none(nenhuma), hidden(não aparece). É possivel tambem escolher o lado da borda (left, top, right, bottom) de forma individual. Há tambem a borda arredondada (radius) e passar umaporcentagem de arredondamento.

  • Margin: serve para dar distançia, espaço embranco em uma das direções do elemento. Também pode-se específicar o lado.
    Filho com margens diferentes para cada lado.

  • padding: da espaçamento também só que do lado de dentro do elemento. É possível também específicar o lado e fazer short-hand (top right bottom left). O padding é muito utilizado para criar botôes.
    Div sem o padding

    Div com padding top 10, bottom 10, left 20, hight 20;


  • width: largura do elemento. A max-widht: determina a altura máxima que esse elemento pode chegar;


  • height: altura;


  • text: text-align (alinhamento de texto - end, start, center, justify...), direction (rtl - letras invertidas), unicode-bidi (texto detras para frente), text-decoration (geralmente para estilização de link - none, overline(linha sobre), line-trough(riscado), under-line(sublinhado)), text-tranform (uppercase, lowercase, captalize), text-ident (recuo de texto - px), letter-spacing (espaçamento entre letras - negaticos e positivos - px), line-heigt(altura das linhas - em decimais), word-spacing( espaçamento entre palavras - px).


  • font: font-family(família da fonte), font-size(tamanho da fonte - px); è possível colocar fontes externas, para isso use o Google fonts.
