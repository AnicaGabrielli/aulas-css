o float é nescessário para quando se precisa que um conteúdo vá para posições específicas dentro do container pai.

o elemento com propriedade float fica como flutuante no container pai. Para que o container pai entenda que deve pegar todo o elemento com o float, deve se cololocar a propridedade overflow como auto. Se quiser fazer com que um título, por exemplo, não precise espera o elemnto float acabar, para ocupar a posição, coloca-se a propriedade clear com aributo both;
